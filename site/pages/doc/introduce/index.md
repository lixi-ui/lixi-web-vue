### 介绍
### web 的展示
### npm install lixi-cli -g

### 需要安装依赖包。template 为脚手架安装的目录

### 安装 web 项目, 开发的业务代码
### lixi-cli create lixi-demo vue-web

### web结构目录解析

```sh
  |- web
    |- material   --------------------------------- 本地材料库的入口
      |- index.js  -------------------------------- 本地材料库的入口文件
    |- public  ------------------------------------ 公共文件
      |- index.html ------------------------------- 公共文件页面
      |- lixi-logo.png ---------------------------- logo图片
    |- script
      |- webpack.build.js ------------------------- 打包生成环境文件
      |- webpack.web.js --------------------------- 启动项目文件
    |- server ------------------------------------- node服务
      |- bin -------------------------------------- node 启动文件
        |- www ------------------------------------ 启动文件
      |- public ----------------------------------- 公共路径
      |- router ----------------------------------- 路由控制
      |- view ------------------------------------- 模板模型
      |- app.js ----------------------------------- app入口 
    |- site --------------------------------------- 前端代码
      |- assets ----------------------------------- 资源文件
      |- components ------------------------------- 组件文件
      |- md-loader -------------------------------- 解析md-loader
      |- pages ------------------------------------ 页面
      |- router ----------------------------------- 路由配置
      |- style ------------------------------------ 样式
      |- unit ------------------------------------- 单元代码
      |- app.vue ---------------------------------- 前端 vue 入口文件
      |- index.js --------------------------------- 前端入口文件
    |- .babelrc
    |- .gitignore
    |- jest.config.js
    |- LICENSE
    |- package-lock.json
    |- package.json
    |- README.md
```
